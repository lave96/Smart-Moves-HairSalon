# Smart Moves Hair Salon

## Project Overview
Smart Moves Hair Salon is a modern, professional salon catering to both Ladies and Gents in the heart of Islamabad. We provide premium hair styling, makeup, nail art, and bridal services with a commitment to excellence and customer satisfaction.

## Features
- **Premium Services**: Hair styling, makeup, nail art, bridal packages, and grooming
- **Ladies & Gents**: Separate facilities and expert staff for all clients
- **Online Booking**: Easy appointment scheduling
- **Professional Team**: Experienced stylists and beauty experts

## Tech Stack
- **Frontend**: React.js, TypeScript, Tailwind CSS
- **Build Tool**: Vite
- **Animations**: Framer Motion

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open your browser at `http://localhost:3000`

## Deployment
The app is configured for easy deployment to Vercel, Netlify, or any static hosting platform.
